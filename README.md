## 📖 Overview

**Brain Quest** (deployed as *Locked in Reality 2026*) is a fully type-safe, competitive escape room platform designed for hackathons, educational events, and programming competitions. Teams race against a **2-hour countdown** to solve eight cryptographic and logic-based puzzles embedded within a retro terminal aesthetic — think green phosphor text, CRT scan lines, and glitch animations.

A standout mid-game event called the **"Mainframe Break"** introduces a surprise one-attempt bonus challenge, adding unpredictability to the competition.
